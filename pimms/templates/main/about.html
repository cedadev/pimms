What is PIMMS?

PIMMS provides a method for consistent and comprehensive documentation of modelling activities that enables the sharing of simulation data and model configuration information. The aim of PIMMS is to package the metadata infrastructure developed by Metafor for CMIP5 so that it can be used by climate modelling groups in UK Universities.

1. Typical Workflow

In generating a project-specific metadata questionnaire three primary resources need to be in place before this can be done. These are:

  • One or more 'Numerical Experiment' documents
  • One or more 'Controlled vocabularies' documents
  • A number of project level settings that are define a particular metadata questionnaire implementation

2. Creating a Numerical Experiment

A numerical experiment describes an 'imagined test' that is used to investigate a scientific hypothesis and defines the constraints associated with such a hypothesis. A numerical experiment lays the foundation for simulations to be run against it. In PIMMS, a numerical experiment is composed of some top-level information describing the experiment. This includes such information as name, description and experiment rationale. The constraints imposed within an experiment are defined within a list of associated 'numerical requirements', and can pertain to initial, boundary and/or spatio-temporal conditions.
